    # 实数

    了解实数的分类,数轴,相反数和倒数、绝对值、算数平方根的概念及其有关计算.

    # 实数

    • 有理数 可以整除或者有规律的循环
      • 整数
        • 正整数(自然数)
        • 零(自然数)
        • 负整数
      • 分数
        • 正分数
        • 负分数
    • 无理数 无限不循环小数(π,开根开不尽,e 自然数)
      • 正有理数
      • 负有理数

    # 实数的基本概念

    1. 数轴 规定原点、正方向、单位长度的直线叫做数轴. #----------0----1--------->
    2. 相反数 绝对值相同而符号相反的两个数,互称为相反数. 一对相反数的和等于 0(|a| |-a|)
    3. 倒数 一除以某数的商我们叫做倒数,零没有倒数. a-> 1/a a 的倒数为 1/a a 与自己的倒数相乘 =1 a*(1/a)=1 (1/a)*a=1
      -3 的倒数 是 -(1/3)
    4. 绝对值 正数绝对值本身,负数绝对值是相反数.(本身到 0 的距离)

    # 实数的运算

    原数 同号 异号
    运算 符号 绝对值 符号 绝对值
    加 保持原号 相加 同绝对值较大的符号 相减
    减 减去一个数等于加上这个数的相反数
    乘 + 相乘 - 相乘
    除 + 相除 - 相除
    乘方 正数的任何次幂都是正数;负数的偶次幂是正数,奇次幂是负数;零的正数此幂等于 0;任何不等于 0 的数的 0 次幂都等于 1;任何不等于 0 的负数-p(p 是正整数)次幂,等于这个数 p 次幂的倒数
    开方 一个正方形有两个平方根,他们互为相反数;0 的平方根是 0;负数没有平方根;一个正数有一个正的平方根;一个负数有一个负的立方根;0 的立方根是 0;
    • 加减法
      • 加法:
        • 法则:
          1. 同号两数相加,取相同的符号,并把绝对值相加. (+2++3=+5) (-2+-3=-5)
          2. 异号相加(绝对值不相等的两个数)取绝对值较大的符号,并用较大的绝对值减去较小绝对值的值(3-2)=1
          3. 如果两个数互为相反数,互为相反数的两个数相加值为 0;
          4. 任何数与 0 相加,仍得这个数;
      • 减法
        • 法则
        1. 减去一个数等于加上这个数的相反数 (5-4) = (5+-4)
      • 运算率
        1. 交换律 :两个数相加 交换先后顺序,结果不变(a+b) = (b+a)
        2. 结合律 :三个数相加先把前两个数相加或者先把后两个数相加和不变(a+b+c) = a+(b+c)
      • 混合运算(有理数的混合运算 优先考虑的事项)
        1. 如果两个数互为相反数,优先相加
        2. 如果几个数的和为正数,优先相加
        3. 分数相加,把同分母的数相加
        4. 同符号的数相加
        5. 最后相减

    • 乘除法
      • 法则
        • 两数相乘,同号得正,异号得负,并把绝对值相乘;(33)=正 (-2-3) 正 (4*-3) 负

        • 任何两个数的乘积为 1,我们称 互为倒数; 乘积为 1 的数互为倒数;
      • 多个有理数不包含 0
        • 相乘取决于负因数的个数,为奇数即为负,当负因数的个数为偶数时,即为正;
        • 积的绝对值等于个因数的绝对值的积

        • 分配律:一个数同两个数的和的积,等同于吧这个数分别与两个数相乘在吧积相加;

    # 整式

    单项式跟多项式统称为整式

    # 单项式
    1. 表示数或者字母乘积的式子
    2. 或者是一个字母,也是单项式 /a,2a/2ab
    3. 单项式中的数字因数我们称为单项式的系数; (2ab2,系数 2,次数 3)
    4. 在一个单项式中所有的字母的指数的和叫作单项式的次数;
    5. 圆周率派式常数 (-3πab,系数-3π,次数 2) 单项式的系数与次数;
    # 多项式
    1. 几个单项式的和叫作多项式;
    2. 每个单项式也称为多项式的项;
    3. 不含字母的项称为常数项;(2x+3x-1)-1 不含字母 常熟项
    4. 多项式的次数指的是,多项式中次数最高项的次数,
    5. 根据多项式中次数与项数可以称一个多项式为 x 次 x 项式 -3 的 7 次方-2 = 7 次 2 项式
